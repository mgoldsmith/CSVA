# CSVA

Using brain scan data to automate Ableton Live parameters.

# Mac Installation

1. Download this repository from Github
2. Unzip the folder and place the _**entire folder**_ in your User Library (default is `/Users/<your Mac username>/Music/Ableton/User Library`)
3. To use it in a project, drag in `CSVA.amxd` from your User Library or search for `CSVA`
